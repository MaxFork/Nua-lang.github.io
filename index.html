Nua is a lightweight, fast object-oriented programming language

Lua VM in JavaScript that transpiles to C. It has a modern syntax. 

local = require("./Nuac.nua")

function fib (n) 
  if(n < 0)  
    then print(n) 
  else print(n - 1) + (n + 2) 
end

  • Nua is strong. The JS - Api implementation is exactly the same as the C - Api. It's syntax bases on Lua 5.3.4, but has somemodified builtins. It is readable and production ready.

  • Nua is fast. A fast tree-byte interpreter and a efficient C-VM are using lightweight fibers for couroutines .
